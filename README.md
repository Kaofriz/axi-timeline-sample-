About
This timeline is a companion resource to:

AXI (Draft v0.3, 2026) — A Philosophy of Difference, Dimension, and Local Optimisation
AXES: Axes of Epistemic System (Draft v0.6, 2026.03.28) — Zenodo DOI: 10.5281/zenodo.19264332

The timeline places thinkers and philosophical traditions from across cultures and centuries alongside their historical context, and maps each in relation to AXI — identifying points of resonance, effective domains, and divergence.
Note: This is a sample version for the AXI paper. A fuller interactive version is in development.

How to use

View ① — Simple: thinkers and context, no tags
View ② — Tags + AXI detail: click any row to expand the AXI comparison panel (resonance / effective domain / difference)
